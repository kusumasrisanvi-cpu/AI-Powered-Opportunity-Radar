# AI Powered Opportunity Radar

AI Powered Opportunity Radar is an AI-powered web application that helps students and job seekers discover personalized internships, jobs, scholarships, hackathons, and career opportunities based on their skills, interests, education, and career goals.

## Features
- AI-based opportunity recommendations
- Resume analysis
- Application tracker
- Deadline reminders
- Personalized dashboard

## Tech Stack
- React + Vite
- Node.js + Express
- PostgreSQL
- OpenAI API
- Tailwind CSS
