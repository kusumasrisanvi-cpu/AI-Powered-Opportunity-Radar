#  AI Powered Opportunity Radar — Scaffold

> Generated by iNSIGHTS AI Project Architect

## Tech Stack

- **React + Vite**: Frontend framework
- **Node.js/Express**: Backend API
- **PostgreSQL**: Database
- **OpenAI API**: AI Engine
- **Tailwind CSS**: Styling

## Folder Structure
```
root/ (src/api, src/components, src/hooks, src/utils, src/pages, public/, tests/)
```

## Database Schema (DDL)
```sql
CREATE TABLE users (id UUID PRIMARY KEY, email VARCHAR UNIQUE, skills JSONB); CREATE TABLE opportunities (id UUID PRIMARY KEY, title VARCHAR, deadline TIMESTAMP);
```

## API Endpoints

### `POST /api/auth/login`
Authenticate user

### `GET /api/opportunities/recommend`
Get AI-matched opportunities

### `POST /api/resume/analyze`
Analyze resume against job description
